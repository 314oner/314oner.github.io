
\* Accepted fields:

| Option    | Type                             | Default       | Description                |
| --------- | -------------------------------- | ------------- | -------------------------- |
| `onClick` | `MouseEventHandler<HTMLElement>` | null          | Element click handler      |
| `success` | `boolean`                        | `true`        | Set success display        |
| `typical` | `boolean`                        | `false`       | Set typical behavior       |
| `size`    | `string`                         | `'undefined'` | Set the size of the button |
