```
import { RefButton } from '@314oner_npm/universal-components-library';
```

\* Accepted fields:

| Option    | Type                                        | Default     | Description                   |
| --------- | ------------------------------------------- | ----------- | ----------------------------- |
| `success` | `boolean`                                   | `true`      | Set the success of the button |
| `typical` | `boolean`                                   | `false`     | Set the typical of the button |
| `onClick` | `MouseEventHandler<HTMLElement>`            | `null   `   | Element click handler         |
| `size`    | <code>'small' \| 'medium' \| 'large'</code> | `undefined` | Set the size of the button    |
| `color`   | <code>'primary' \| 'dark'\| 'blue'</code>   | `'primary'` | Set the color of the button   |

Open [`http://localhost:3000/universal-components-library/RefButton`](http://localhost:3000/universal-components-library/RefButton)
